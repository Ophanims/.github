# Ophanim Project

欢迎来到 **Ophanim** 开源工程项目。我们目前专注于开发一款高保真的卫星星座模拟系统，旨在为航天任务调度和资源管理提供一个实用的算法验证平台。

### 我们正在做什么？
目前的核心目标是构建 **Ophanim Core**，一个模块化的仿真框架：
- **轨道仿真**：基于物理定律（SGP4/J2）的卫星运行预报。
- **几何计算**：计算卫星载荷对地目标的可见性与覆盖。
- **算法验证**：提供标准接口，让开发者可以测试自己的任务规划策略。
- **态势可视化**：使用 Cesium.js 展示星座运行的 3D 效果。

### 技术栈
- **后端**: Python (FastAPI, Skyfield)
- **前端**: Next.js (Cesium.js)
- **数据**: CZML, SQLite

### 当前状态
本项目目前处于 **MVP (最小可行性产品)** 开发阶段。我们正在完善核心传播引擎与基础的可视化链路。

这是一个开放的工程项目。如果你对轨道动力学、遥感调度或 Cesium 可视化感兴趣，欢迎通过 Issue 提出建议或参与讨论。
